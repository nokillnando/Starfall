Getting Started

Use the following template for your main Starfall file:

--@name Red Fox Engine v3 Car
--@shared
--@include red_fox_engine/simulated_engine_core_modular.txt

dofile("red_fox_engine/simulated_engine_core_modular.txt")
local Core = _G.RedFoxEngineCore

local config = {
    -- Paste variables from below here
    Engine_Redline = 9000,
    -- ...
    Addons = {}
}

if CLIENT then Core.initClient(config) else Core.initServer(config) end

1. Engine Specifications

Variable Default Description
Engine_Bore 87 Cylinder bore (mm).
Engine_Stroke 84 Piston stroke (mm).
Engine_Cylinders 4 Number of cylinders.
Engine_Airflow 135 Airflow (CFM). Used for torque calc if PeakTorque unset.
Engine_Idle 1200 Target Idle RPM.
Engine_IdleRev 0.1 Throttle nudge to maintain idle.
Engine_RevLimiter 1 1 = Hard Cut, 0 = Soft/None.
Engine_RevLimitRPM 9000 Exact RPM to trigger bouncing.
Engine_RevLimitDrop 250 RPM to drop when hitting limiter.
Engine_RevLimitDelay 0.0 Delay before throttle returns (ignition cut).
Engine_Redline 9000 Max RPM.
Engine_FlywheelMass 8 Inertia (kg). Higher = slower revs.
Engine_Configuration 0 0 = Inline, 1 = VPhape (Affects sound/physics hint).
Engine_TorqueTable {1.45, 2.56...} N-Point Bezier Control Table (0% to 100% Redline).
TorqueMultiplier 1 Global final torque scalar.

2. Transmission & Gearing

Variable Default Description
Gearbox_Ratios {3.13...} Forward gear ratios.
Gearbox_FinalDrive 4.10 Final drive ratio.
ReverseGearRatios {-3.58, -2} Reverse gear ratios.
Gearbox_Delay 250 Shift time (ms).
Gearbox_ClutchSpeed 1 Clutch engagement speed scalar.
WheelRPMToSpeed 0.15 CRITICAL: Wheel RPM to KPH factor.
PowerDistribution 0 0=RWD, 50=AWD, 100=FWD.
GearSpeeds {29, 55...} Shift-points (KPH) if UseRPMBasedGear=false.
ReverseGearSpeeds {15, 30} Reverse shift-points (KPH).

3. Transmission Tuning

Variable Default Description
AutoGear true Enable automatic transmission.
ClassicAuto false True = P/R/N/D Logic. False = Sequential R/N/1.
UseRPMBasedGear true Recommended: Use RPM thresholds for shifts.
ImmediateReverseEngage true If true, allows R engagement without full stop.
DriftSlipTolerance 0.50 Allowed slip % before holding gear.
Min_Gear_Time 250 Min time (ms) in gear before shifting again.
Min_Reverse_Gear_Time 150 Min time (ms) in reverse gear.
SpeedToReverse 5 Max KPH to allow engaging reverse.
ReverseWSonReverse true True: W=Back, S=Brake in R. False: S=Throttle.

4. Manual & Shifting Mode

Variable Default Description
ManualModeEnabled true Allow key toggle for manual.
ManualModeKey "n" Toggle key.
AlwaysSportMode false Force aggressive shifts always.
UseComfortSportMode true Toggle shift points based on Mode.
ComfortSportModeKey "m" Key to toggle Sport Mode.
ComfortUpshiftRPM 45 % Redline upsift (Comfort).
ComfortDownshiftRPM 25 % Redline downshift (Comfort).
SportUpshiftRPM 90 % Redline upsift (Sport).
SportDownshiftRPM 55 % Redline downshift (Sport).
DownshiftCooldown 0.35 Seconds between downshifts.
UseClutchLerpInManual true Smooth clutch in manual?
ClutchLerp 0.4 Clutch smoothing factor.
ClutchLerpReverse 0.5 Clutch smoothing factor (Reverse).

FirstPhaseBrakeF/R 0.9 / 0.7 Phase 1 Brake Force (Front/Rear).
SecondPhaseBrakeSpeed 40 Start of Phase 2 (Medium Speed).

ThirdPhaseBrakeF/R 0.3 / 0.05 Phase 3 Brake Force (Front/Rear).

Fine-tune how engine pitch/volume reacts to RPM.

Engine Loop

Variable Default Description

Engine_SoundOnPitchMult 1 Pitch Scalar.
Engine_SoundOnVolume 1 Max Volume.
Engine_SoundOffDiv 3500 Coast Pitch Divisor.
Engine_SoundOffVolume 0.5 Coast Volume.
Engine_SoundBlendRate 8 Crossfade Speed.
Engine_SoundThrottleExponent 1.2 Response Curve.

Turbo & Effects
